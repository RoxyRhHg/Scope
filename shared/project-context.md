# Scope 项目上下文

## 项目是什么

A 股价值投资筛选系统。帮助个人投资者按段永平+巴菲特框架筛选股票，提供买入卖出参考区间和安全边际。

## 核心功能

1. **今日热点**：展示主力资金流入的概念/行业/题材，每个题材下展示推荐股的买入区间和安全边际
2. **重点关注**：价格在买入区间内、安全边际≥10%、估值低估的股票
3. **总榜 Top50**：全市场按 Buffett 核心价值分排名
4. **涨停预测**：十日内最可能涨停的股票（技术+动量+行业热度评分）
5. **ETF/LOF**：场内可交易基金

## 技术架构

- 前端：单文件 `public/start.html`（自包含 HTML/CSS/JS）
- 后端：Node.js `src/server.js`（端口 4173）
- 数据：新浪行情 + BaoStock 行业 + 新浪概念
- 评分：段永平哲学融合版（6维度核心价值分）

## 数据流

```
新浪行情 API
    ↓
realDataAdapter.js (适配为内部模型)
    ↓
ranking.js (评分 + 排名 + 估值)
    ↓
dashboard.js (组装页面数据)
    ↓
server.js (/api/dashboard 返回已评分数据)
    ↓
start.html (前端渲染)
```

## 决策记录

### 2026-06-04 前端重构
- **决策**：将前端从多文件 ES module 改为单文件自包含
- **原因**：旧 app.js 依赖链太长，浏览器静默失败无法调试
- **结果**：start.html 自包含所有 CSS+JS，加载失败有明确错误提示

### 2026-06-04 搜索框中文修复
- **决策**：用 compositionstart/end 处理 IME，而非在 input 事件中处理
- **原因**：input 事件在中文输入过程中会中断 IME 组合
- **结果**：中文输入流畅，不再被打断

### 2026-06-04 API 评分数据
- **决策**：在 /api/dashboard 端点中对数据做评分后再返回
- **原因**：前端需要 scores/buyZoneLow/marginOfSafety 等字段，原始数据没有
- **结果**：前端直接使用扁平字段，无需客户端计算
